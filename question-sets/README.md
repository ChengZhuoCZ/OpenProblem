# Question Sets

Place question set files here.

## Current Collections

- `open-problems/`: Mathematical open problems. Each problem has a `problem.md` statement and a `progress.md` tracker.

Recommended grouping examples:

```text
question-sets/math/grade-9/
question-sets/english/cet4/
question-sets/computer-science/data-structures/
question-sets/open-problems/02-fully-nonlinear-pde/problem-003/
```
