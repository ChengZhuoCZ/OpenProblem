# Problem 013 - Removing Condition (R) in Fourier Restriction to Hypersurfaces

## Source

This is Problem 8 from "Some Problems in Harmonic Analysis", contributed by Detlef Muller.

## Problem Statement

For finite type hypersurfaces \(S\subset\mathbb R^3\), determine whether the known Newton-polyhedron description of Stein-Tomas type restriction estimates remains valid without condition (R).

The source proposes the toy model

\[
\phi(x_1,x_2)=(x_2-x_1^m)^n+\varphi(x_1),
\]

where \(m,n\ge 2\) and \(\varphi\) is smooth and flat at the origin. Does the critical restriction exponent predicted by the unperturbed model remain valid for sufficiently small neighborhoods?

## Background

Ikromov-Muller theory gives sharp ranges under condition (R), and all real-analytic surfaces satisfy that condition. Flat perturbations are a natural test for the smooth finite type case outside condition (R).

## Known Context

- The unperturbed model has adapted coordinates and a computable critical exponent.
- A different perturbation inside the main power may still satisfy condition (R), while the additive flat perturbation generally does not.

## Data Still Needed

- Record the exact critical exponent formula in terms of \(m,n\).
- Check subsequent work on smooth finite type restriction without condition (R).
