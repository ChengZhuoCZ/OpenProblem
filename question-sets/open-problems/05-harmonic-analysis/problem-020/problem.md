# Problem 020 - Hardy Space Endpoint for Lacunary Maximal Functions

## Source

This is Problem 15 from "Some Problems in Harmonic Analysis", contributed by James Wright.

## Problem Statement

Under the Fourier decay assumption on \(\mu\) from Problem 019, determine whether the associated lacunary maximal operator maps

\[
H^1(\mathbb R^d)\to L^{1,\infty}(\mathbb R^d).
\]

## Background

This endpoint is expected to be more accessible than weak type \((1,1)\) on \(L^1\). The source notes that the result is known when \(\mu\) is surface measure on the unit sphere.

## Known Context

- Christ proved the spherical surface measure case.
- Later extensions cover measures satisfying dimensional and smoothing assumptions near \(p=1\).
