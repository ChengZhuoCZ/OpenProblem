# Problem 019 - Weak Type \((1,1)\) for Lacunary Maximal Functions

## Source

This is Problem 14 from "Some Problems in Harmonic Analysis", contributed by James Wright.

## Problem Statement

Let \(\mu\) be a compactly supported finite Borel measure on \(\mathbb R^d\), with dyadic dilates \(\mu_k\), and define

\[
Mf(x)=\sup_{k\in\mathbb Z}|f*\mu_k(x)|.
\]

Assume the Fourier decay condition

\[
|\widehat\mu(\xi)|\le C|\xi|^{-\varepsilon}
\]

for some \(\varepsilon>0\). Does \(M\) satisfy a weak type \((1,1)\) bound?

## Background

\(L^p\)-boundedness is known for \(1<p\le\infty\), but the endpoint weak \((1,1)\) question is open for singular measures satisfying Fourier decay.

## Known Context

- The problem concerns lacunary dilations; more general lacunary sequences can replace \(2^k\).
- The source notes the lack of both counterexamples and positive examples in the singular measure setting.
