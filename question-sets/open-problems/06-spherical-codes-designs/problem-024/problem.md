# Problem 024 - Classification of Extremal \(s\)-Distance Sets on Spheres

## Source

This is taken from Eiichi Bannai's G2C2 Lecture No. 1 on spherical \(t\)-designs.

## Problem Statement

Let \(X\subset S^{n-1}\) be an \(s\)-distance set, meaning the set of pairwise distances among distinct points has size \(s\). It is known that

\[
|X|\le {n-1+s \choose s}+{n-1+s-1 \choose s-1}.
\]

Classify the cases where equality holds.

## Background

The lecture notes that equality examples are known for \((s,n)=(2,6)\) and \((2,22)\), while the equality classification is still open for \(s=2\). For \(s\ge 3\), nonexistence of equality cases is known.

## Known Context

- For \(s=1\), the extremal case is the regular simplex with \(|X|=n+1\).
- The smallest open dimension mentioned for the \(s=2\) equality problem is \(n=118\), after nonexistence in some smaller candidate dimensions.
